Terracotta is a large-only text color.

Fired Terracotta as text on linen scores 3.06 (3.37 on bone) — AA-large only. Use it for the one accent word in a headline, big numbers, or links ≥24px. Never for body or small UI text. This is the Color Spike Rule, now with a number.

!
Caption Ash (steel-400) is for large or non-essential text.

On bone it scores 4.23 — just under AA-normal. Fine for hints, timestamps, and ≥16px captions; for body-critical small text, step up to Body Smoke (steel-500, 8.74).

!
Prefer the soft success badge for status text.

Linen on the Turf Green solid is 3.79 — below AA for small badge text. Use the solid green for icon chips and large fills; for a text status, use the soft variant (deep green on wash, 8.58).
